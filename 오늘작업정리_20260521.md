## 3. 오늘 수정한 핵심 버그 기록

| 버그 | 원인 | 해결 방법 |
|------|------|-----------|
| Supabase RLS 오류 (42501) | database.py가 anon 키만 사용, JWT 없음 | 로그인 시 session 저장 → get_supabase()에서 JWT 주입 |
| 달력 기존 기록 안 뜸 | 기존 데이터 user_id = NULL → 필터에서 제외 | `or_(user_id.eq.X, user_id.is.null)` 적용 |
| 날짜 클릭 KeyError | dateClick 반환 키가 "dateStr"가 아닐 수 있음 | `.get("dateStr") or .get("date")` 안전 처리 |
| 날짜 클릭 시 -1일 표시 | streamlit-calendar가 UTC 기준 날짜 반환 | KST 보정: 반환 날짜에 +1일 추가 |
