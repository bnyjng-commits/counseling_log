## 2. 확정된 앱 기획

**앱 이름:** AI 상담일지 (중학교 수학 교사용)

### 화면 구성
- 🔐 로그인/회원가입 (Supabase Auth 이메일+비밀번호)
- 🏠 홈 (달력) — 첫 화면, 상담 기록 + 예정 일정 표시
- ✍️ 상담일지 작성 — 음성/카메라/사진 업로드 + AI 분석
- 📂 이력 조회 — 학생별/날짜별/카테고리별 필터
- 📝 생활기록부 문구 생성 — 체크박스 선택 + AI 생성 + 복사

### 기술 스택
- Streamlit + Supabase Auth + Supabase DB + Anthropic API
- 배포: Streamlit Cloud

## 5. 트러블슈팅 기록

| 오류 | 원인 | 해결 |
|------|------|------|
| PowerShell pip 오류 | 실행 정책 제한 | PyCharm 터미널로 변경 |

| Anthropic 인증 오류 | 원인 불명확 | ⚠️ 다음 세션에서 해결 필요 |
